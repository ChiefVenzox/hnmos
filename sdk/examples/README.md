# HNMos SDK Examples

Bu klasor, SDK tarafindan hedeflenen uygulama tiplerinin kucuk orneklerini tutar.

Ornek uygulama aileleri:

- `hn-notes`: not ve metadata odakli basit gorev uygulamasi.
- `hn-monitor`: sistem durumunu ve event snapshot'larini okuyan monitor.
- `hn-ai-lab`: AI Runtime Interface ile izinli deney alani.
- `hn-files`: Knowledge FS gorunumu hedefi.
- `hn-builder`: build ve verify gorev araci hedefi.
- `hn-camera-agent`: camera event stream icin izinli agent hedefi.
- `hn-dev-studio`: HNLang, shell ve task monitor odakli gelistirme hedefi.
