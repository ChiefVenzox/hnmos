# hn-notes

`hn-notes`, HNMos uzerinde calisacak basit not ve metadata uygulamasi hedefidir.

Ilk hedef:

- HNShell uzerinden calistirilabilir olmak.
- Knowledge FS metadata modeliyle not basligi, etiket ve ozet tasimak.
- Dosya yazma icin permission manifest istemek.
- AI tarafindan not ozetleme onerisi alabilmek, fakat kullanici onayi olmadan dosya degistirmemek.

Calistirma hedefi:

```text
hn app run hn-notes
```
